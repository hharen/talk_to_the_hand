How the gesture code works

Windspeeds 1-99 are signed with the back of the hand using up to two gestures: a tens gesture first, then a ones gesture.

  • A single digit (1-9) uses the ones gesture only.
  • An exact ten (10, 20, … 90) uses the tens gesture only.
  • Anything else is the tens gesture, then the ones gesture.
  • The thumb is not used for counting — except for 5.
